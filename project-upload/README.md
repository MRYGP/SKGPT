# ChatGPT Project 上传包快照

本目录是 ChatGPT Project Knowledge 25 个上传文件的本地/GitHub 快照，不是 SK 主仓库的替代品。

目录结构：

- stable-core/：18 个稳定内核文件
- current-mainline/：5 个当前主线资产
- task-slots/：2 个任务机动文件

维护规则：

1. 文件正文来自 `MRYGP/SK` 当前主仓库。
2. 上传清单以 `knowledge/upload-manifest.md` 为准。
3. 动态状态文件不放入本目录长期占位。
4. `CLAUDE系统指令.md` 不放入 25 个上传位。
5. GitHub 中本目录更新后，不会自动同步到 ChatGPT Project，仍需人工在 Project Knowledge 中删除旧文件、上传新文件。
