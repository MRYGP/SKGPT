# 「诊断空白」赛道拆解库

> 创建：2026-04-14
> 更新：2026-04-20（新增姐妹框架引用）
> 来源：011号文章（MYHAIR AI）推演过程中发现的横向可复制模式
> 核心命题：**"用户现在要不要行动"——这道没被解的题，是一类市场机会的通用入口**
> 用途：系统性扫描同类市场，找到下一个可以拆解或押注的方向
> 摘要见 `横向模式种子-AI诊断市场通用公式.md`
> **姐妹框架**：`core/SKILL-多维综合产品推演框架.md`（多维综合="持续追踪"的空白，本库="一次性判断"的空白）

---

## 一、这个文档是干什么的

011拆完MYHAIR AI之后，发现了一个可以横向复制的模式：

在很多健康市场里，存在大量"有焦虑但没行动"的用户。
他们不是不在意，而是卡在同一道坎：**不知道自己到底严不严重，迈不出"该不该行动"的那一步。**

这道坎，就是「诊断空白」。

AI自测工具，恰好可以填这个空白。
填了之后，该行动的用户自己会行动，B端专业机构接到的是精准客户。

这个文档用来追踪：哪些市场有同样的结构，值不值得深推。

---

## 二、筛选框架（四条件）

**四个条件同时成立 → 这个市场值得深推：**

| 条件 | 判断问题 |
|------|---------|
| **条件1：高频焦虑** | 这件事每天/每周都会出现在用户感知里吗？ |
| **条件2：低频行动** | 有焦虑的人里，真正寻求专业干预的比例极低（<5%）吗？ |
| **条件3：诊断空白** | "我到底严不严重、需不需要行动"这个问题，目前有没有简单可信的答案？ |
| **条件4：B端导流价值** | 有没有一个专业机构，痛点是"找不到已经准备好行动的人"？ |

## 三、六个市场逐一拆解

### 市场1：脱发（模板案例）

**那个卡住的人：**
30岁男性，每天照镜子数掉发，看了20篇小红书帖子，买了防脱精华，不知道有没有用。想去医院，觉得没严重到那一步。

**诊断空白在哪：**
没有一个工具能在5分钟内，用可信的方式告诉他：
你是轻度（护理产品够用）、中度（需要药物干预）、还是重度（才真的需要植发）。

**现有方案为什么不够：**
- 小红书博主：通用建议，无法针对个人，且利益不中立
- 植发医院初诊：太重（要去一趟，要被销售），且不被信任
- 防脱产品：买了不知道对不对症

**B端机会：**
植发医院销售费用率40-50%，大量预算花在"说服不确定的人"上。
AI自测筛出"已经确定需要干预"的用户，精准线索价值500-1500元/个。

**AI可行性：** ✅ 拍照→视觉分析，技术成熟，已有海外验证（MYHAIR AI）

**监管包装：** "脱发风险自测" / "头发健康评分" ≠ 医疗诊断

**中国竞品现状：**
尚无成熟的独立消费级AI脱发自测产品（现有方案均为B端硬件或机构内部工具）

**需要的合伙人：**
有植发/医美行业BD资源，能走进雍禾/碧莲盛谈合作的人

**时间窗口：** 12-18个月
**优先级：** ⭐⭐⭐⭐⭐（已深推，011文章已写）
**四路决策：** 工具（招募有行业资源的合伙人）

### 市场2：皮肤（痘/色斑/敏感肌）

**那个卡住的人：**
25岁女性，长了一些痘，有几块色斑，皮肤有时候泛红过敏。用了大量护肤品，看了无数小红书，每隔几个月买一批新产品，不见好转。想去皮肤科，但觉得"就这点问题，去医院是不是小题大做"。

**诊断空白在哪：**
不知道自己皮肤问题的根本原因是什么（激素？屏障受损？过敏源？）
不知道自己现在靠护肤品能不能解决，还是真的需要医疗干预。
不知道市面上哪些产品对自己的"具体问题"有用，哪些在浪费钱。

**现有方案为什么不够：**
- 护肤品牌的"测肤"：利益不中立，目的是推荐自家产品
- 小红书护肤博主：通用经验，无法对应个人具体问题
- 皮肤科：很多人觉得"不够严重"不好意思去，或者挂号麻烦

**B端机会：**
皮肤科医院、医美机构的获客成本高（竞争激烈，医美广告费用高）
AI皮肤评估筛出"已经到达需要专业干预程度"的用户，对诊所的价值非常直接
护肤品牌也愿意为"精准匹配对症用户"付费（精准推荐可以提升转化率、降低退货率）

**AI可行性：** ✅ 拍照→皮肤问题识别，技术已相当成熟（市面已有类似产品）

**监管包装：** "肤质评估" / "皮肤健康分析" ≠ 医疗诊断

**中国竞品现状：**
有一定竞品（美图/完美日记等做过测肤功能），但多数定位是"推荐产品"而非"诊断问题"。
真正做"帮你搞清楚自己皮肤问题根本原因"的独立中立工具，空白。

**需要的合伙人：**
有皮肤科/医美行业资源，或护肤品牌供应链资源

**时间窗口：** 6-12个月（竞争开始活跃）
**优先级：** ⭐⭐⭐⭐（强，但竞品更多）
**四路决策：** 待深推后决定

**待补充数据：**
- 中国皮肤科/医美市场规模
- 现有"测肤"产品的用户数据和口碑
- 医美机构的线上获客成本

### 市场3：糖尿病前期

**那个卡住的人（真实原型）：**
50岁男性，六年前确诊2型糖尿病，靠运动和节食控制，长期不吃主食，血糖读数就正常，觉得"自己有研究能管住"，没有系统随访。最近体重增加4公斤，空腹血糖7.7+，眼睛出现不适。面对这些信号，还在想"要不要去医院"——不是不在意，是不确定"自己是否已经严重到需要专业干预"。

**诊断空白在哪：**
- 血糖6-7之间（临界区），不知道要不要管
- 靠自己运动节食控制，数字好看，但不知道是否真的在改善（HbA1c才是真实报告）
- 网上关于主食要不要吃的信息混乱，各说各话
- 不知道眼睛不适和血糖有没有关系
- "要不要找中医调理"和"要不要去内分泌科"之间，无从判断

**现有方案为什么不够：**
- 医院内分泌科：挂号繁琐，很多人觉得"还没那么严重"
- 各类健康APP：大多提供通用建议，缺乏个性化追踪和判断
- 网络信息：混乱且相互矛盾，反而增加焦虑而非减少

**B端机会：**
内分泌科、慢病管理门诊、营养师机构、中医调理机构
这些机构共同的痛点：患者来的时候往往已经很严重了，错过了早期干预窗口
精准识别"血糖偏高且开始出现并发症迹象"的用户，导流价值极高

**AI可行性：** ⚠️ 中等
- 不能靠纯图像判断（血糖需要数值）
- 可以靠用户自报数据（血糖仪读数+症状问卷+饮食记录）做风险评估
- 技术路径：数据输入型，非图像识别型

**监管包装：** "血糖管理记录" / "健康风险评估" / "慢病自我监测" ≠ 医疗诊断

**中国竞品现状：**
有一些慢病管理APP（如糖大夫、掌上糖医等），但多数是"管理已确诊患者"，
针对"前期和临界人群"的专项工具相对空白。

**需要的合伙人：**
内分泌科医生/营养师背景，提供医学背书
或慢病管理机构，有患者资源

**额外门槛：** 医疗数据合规（需要处理血糖等健康数据），要做好隐私保护

**时间窗口：** 12-18个月
**优先级：** ⭐⭐⭐⭐⭐（人群最大，机会最大，但门槛最高）
**四路决策：** 待深推后决定

**真实洞察（来自推演者本人）：**
"不吃主食血糖就控制在6以下"——这是控制了测量数字，不是改善了疾病本身。
靠极端饮食控制出来的好数字，会掩盖真实的健康状态。
HbA1c（糖化血红蛋白）才反映过去三个月的真实控糖情况，靠不吃主食骗不了它。
**这个洞察本身就是这个产品的核心价值：帮用户看到真实状态，而不是被数字欺骗。**

### 市场4：脂肪肝

**那个卡住的人：**
40岁男性，体检查出"轻度脂肪肝"，医生说"注意饮食和运动"，然后就没有然后了。他知道自己有问题，但完全不知道轻度脂肪肝到底代表什么，需不需要专门干预，怎么知道自己在变好还是变差。下一次体检又是一年后。

**诊断空白在哪：**
- 轻度/中度/重度脂肪肝意味着什么，风险差多少
- 不知道自己是否在改善（没有可信的日常追踪方式）
- 不知道"注意饮食和运动"具体应该怎么做，做了有没有用

**现有方案为什么不够：**
- 年度体检：一年才一次，中间完全没有信号
- 消化内科：大多数人觉得"没症状，不需要去"
- 各类减肥/养肝APP：通用建议，不针对脂肪肝程度

**B端机会：**
消化内科、体重管理门诊、功能医学机构
脂肪肝是很多代谢综合征的早期信号，早期干预价值高
精准找到"轻度脂肪肝但需要改变生活方式"的用户，机构的干预效果更好

**AI可行性：** ✅ 中高
- 用户输入体检报告数据（ALT/AST/甘油三酯等）+ 问卷
- AI解读指标含义，给出个性化的"你现在的状态和趋势"
- 不需要图像，数据解读型

**一个特别的切入点：**
中国每年体检量巨大，大量人拿到体检报告不知道怎么看。
"帮你看懂体检报告里的脂肪肝相关指标"，是一个更宽的入口，
脂肪肝只是其中一个切入点。

**监管包装：** "体检报告解读" / "代谢健康追踪" ≠ 医疗诊断

**时间窗口：** 12-18个月
**优先级：** ⭐⭐⭐⭐
**四路决策：** 待深推后决定

### 市场5：口腔健康

**那个卡住的人：**
35岁，牙齿偶尔酸痛，知道好几年没去洗牙了，照镜子觉得牙龈有点红。但觉得"还能用"，一直没去口腔诊所。

**诊断空白在哪：**
- 不知道自己的牙龈状态是正常还是已经有牙周病前兆
- 不知道现在去洗牙是必要的还是可以再等
- 被口腔诊所"过度治疗"的说法吓到，不知道如何判断建议是否合理

**现有方案为什么不够：**
- 口腔诊所：信任度问题（担心过度治疗）
- 自己判断：完全没有参照系

**B端机会：**
口腔连锁诊所获客成本不低，洗牙是极高频的引流项目
问题在于：来了不知道后续是否真的有问题，转化率不稳定
AI口腔初评筛出"真的有牙周问题需要处理"的用户，诊所的服务效率和信任度都提升

**AI可行性：** ✅ 中等
拍照识别牙龈颜色/肿胀情况，技术可行
但口腔内部照片质量控制有难度

**额外价值：** 解决"信任问题"——中立的AI评估，帮用户判断诊所建议是否合理

**监管包装：** "口腔健康自查" / "牙龈状态评估" ≠ 医疗诊断

**时间窗口：** 12-18个月
**优先级：** ⭐⭐⭐
**四路决策：** 待深推后决定

### 市场6：视力（干眼症/老花/近视加深）

**那个卡住的人：**
45岁，最近眼睛容易疲劳，看近处有些模糊，偶尔干涩。用了几种眼药水，效果有限。觉得可能需要换眼镜了，但不知道现在是否严重到需要去医院检查，还是买副老花镜就够了。

**诊断空白在哪：**
- 干眼症、近视加深、老花，症状相似但处理方式完全不同
- 不知道自己是哪种情况
- 不知道现在到了需要专业干预的程度，还是调整生活习惯就够

**现有方案为什么不够：**
- 眼镜店：利益不中立（希望你配眼镜）
- 眼科医院：大多数人觉得"没严重到那一步"
- 眼药水：治标不治本，且很多人用错了类型

**B端机会：**
眼科诊所、视光中心、干眼症专科
近视防控是当前政策重点，眼科赛道热
精准找到"已经到达需要专业干预程度"的用户，导流价值清晰

**AI可行性：** ✅ 中高
视力自测（视力表数字化）+ 干眼症状问卷 + 眼部疲劳追踪
技术路径成熟，已有部分产品在探索

**监管包装：** "视力健康自测" / "眼部疲劳评估" ≠ 医疗诊断

**时间窗口：** 12-18个月
**优先级：** ⭐⭐⭐

## 五、深推触发条件

**以下任一条件满足，启动该市场的深度推演：**

1. 有合伙人/资源伙伴在该领域出现（最直接的触发器）
2. 发现海外有成熟的对标产品（说明需求已验证）
3. 该市场出现融资信号或竞品上线（时间窗口收窄，需要判断）
4. 内容线的读者反馈中，该方向有明显共鸣（用户投票）
