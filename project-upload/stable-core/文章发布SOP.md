### 5套文案模板（每篇都要准备，用Claude生成初稿，自己改）

**① 朋友圈文案（≤3句话+链接）**
> 公式：一句反直觉的话 + 一句"我写了什么" + 链接
> 例："所有AI产品都在做'照顾老人'。但老人真正要的不是被照顾。第一篇拆解写出来了。[链接]"

**② 微信群引导语（每个群用不同的，≤2句话+链接）**
> 公式：跟这个群相关的切入角度 + "想听你们的看法"
> AI群："写了一篇XX方向的AI产品拆解，发现一个反直觉的结论。想听你们的看法。[链接]"
> 创业群："拆了XX产品，发现小团队能复制的一个打法。[链接]"
> 投资群："拆了XX赛道，时间窗口可能只有6个月。[链接]"

**③ 私信模板（每人定制一句+通用请求）**
> 公式：一句跟他相关的话 + "帮我看看，5分钟能读完"
> 通用版："帮我看看这篇拆解，如果觉得有意思帮我点个在看/推荐。5分钟就能读完。[链接]"
> 定制版（杨写）："XX总，这篇拆的是你之前提到的XX方向，我有个不一样的判断。[链接]"

**④ 评论区首条（自己先发，引导讨论）**
> 公式：一个读者能回答的开放问题
> 例："你身边有没有这样的产品——用了一次觉得不错，但再也没打开过？它缺了什么？"
> 要求：跟文章主题相关，但不能是只有专业人士才能答的问题

**⑤ 即刻帖子（2-3条，当晚发）**
> 帖1（洞察钩子）：文章的核心反直觉观点，≤3句话，不带链接
> 帖2（数据/案例钩子）：文章里最有冲击力的一个数据或案例，≤3句话
> 帖3（引流）：简短总结 + "完整拆解在公众号三湘问道" + 链接

### 知乎改编方法（标准流程）

1. 把原文贴进Claude
2. 提示词："帮我改成知乎问答体。标题用问句（从文章的核心矛盾出发）。加强论证结构，展开数据和案例。去掉招募/引流段落。结尾自然提一句'更多拆解在公众号三湘问道'。"
3. 自己审一遍——去掉AI味重的地方（"值得注意的是""综上所述"），加口语化表达
4. 知乎标题公式：**"为什么XX都做不起来？"** 或 **"XX行业最大的坑是什么？"**

### 小红书6张卡片标准结构

| 卡片 | 内容 | 设计要点 |
|------|------|---------|
| 第1张 | 标题卡：文章核心观点（一句反直觉的话） | 大字+对比色，制造停留 |
| 第2张 | 故事/案例卡：文章里最打动人的那个故事或数据 | 配图或数据可视化 |
| 第3张 | 洞察卡："为什么会这样？"——一句话解释机制 | 简洁，不超过30字 |
| 第4张 | 对比卡：成功vs失败 或 别人vs我们的判断差异 | 左右对比排版 |
| 第5张 | 行动卡：读者看完能做什么（一个简单的自检问题） | 互动感 |
| 第6张 | 引流卡：公众号名+一句话定位 | 固定模板，每篇复用 |

制作方法：用Claude生成6页分镜文案 → Canva搜"小红书模板"套用 → 调色调文字

## 七、每5篇做一次复盘

每发完5篇（001-005、006-010……），做一次数据复盘：

### 复盘模板

**1. 数据对比**
| 指标 | 001 | 002 | 003 | 004 | 005 | 趋势 |
|------|-----|-----|-----|-----|-----|------|
| 阅读量（7d） | | | | | | ↑/↓/→ |
| 推荐数 | | | | | | |
| 新关注 | | | | | | |
| 有效私信 | | | | | | |
| 知乎浏览 | | | | | | |

**2. 三个问题**
- 哪篇传播最好？为什么？（标题/话题/发布时间/哪个渠道贡献最大？）
- 哪篇最差？什么环节掉链子了？
- 有没有意外发现？（比如某个平台突然起量、某类读者突然出现）

**3. 下5篇调整**
- 冷启动策略要不要改？（群/私信名单/时间/文案风格）
- 哪个平台投入产出比最高？要不要加大？
- 哪个平台没效果？要不要暂停？

**4. 飞轮检查**
- 有没有读者开始主动转发/推荐？（飞轮信号）
- 有没有读者用我们的框架（比如"三个问题"）去分析别的产品？（框架传播信号）
- 有没有新的合伙人/资源伙伴/跟投人出现？（三类人转化信号）

## 十、时间预算（实习生A视角）

稳定状态下（第3篇起），每篇文章的传播工作总耗时：

| 阶段 | 耗时 | 备注 |
|------|------|------|
| 发布前准备（排版+首图+5套文案） | 2.5小时 | Claude辅助后可压到2小时 |
| D0发布+冷启动（当晚） | 2小时 | 含守评论区 |
| D+1多平台分发（知乎+Wechatsync） | 2小时 | 知乎改编最耗时 |
| D+2小红书 | 1.5小时 | Canva熟练后可压到1小时 |
| D+3~D+7长尾维护 | 每天15分钟 | 回评论+记数据 |
| **每篇总计** | **约9小时** | 分散在一周内 |

按每周发2篇算，传播工作占实习生A约18小时/周——剩余时间做信息收集和拆解辅助。
